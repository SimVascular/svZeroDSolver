Detailed Description

template<typename T>
class MODEL::BloodVesselJunction< T >

BloodVesselJunction.

Models a junction with one inlet and arbitrary outlets using MODEL::BloodVessel elements between each inlet and outlet pair.

Governing equations

The governing equations are mainly defined by the blood vessel elements (see MODEL::BloodVessel for more information). For the sake of brevity, the blood vessel contributions are not repeated here.
